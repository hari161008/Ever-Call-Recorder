# Screenshots

Зберігайте скріни UI сюди з такими іменами (так на них посилається головний README):

| Файл | Що показано |
|---|---|
| `01-home.jpg` | Список дзвінків (головний екран) |
| `02-playback.jpg` | Плеєр запису (scrubber, channel meters, transport, speed) |
| `03-transcript.jpg` | Структурований транскрипт з тонами і спікерами |
| `04-settings.jpg` | Налаштування — авто-запис, sample rate, формат |

## Рекомендації

- **Розмір**: ~1080×2400 (стандартний Android viewport) або менший — головне прийнятна якість.
- **Формат**: JPG (поточні скріни) або PNG (для нових — точніше, без compression artifacts на UI кантах).
- **Контент**: тестові дані (з debug-only `SeedDataActivity`), щоб не світити реальні номери / контакти.
- **Аноніміз**: жодних реальних номерів телефонів навіть з debug seed — використовуйте `+380XXX1234567` тощо.
